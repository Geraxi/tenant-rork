# 🏠 Tenant - Swipe-Based Rental Platform

## Find your next home faster — swipe, filter & chat

**Skip endless scrolling through rental listings. Tenant brings the intuitive matching experience of modern dating apps to property hunting, making it faster and more enjoyable to find your perfect home.**

*Built with React Native & Expo for seamless cross-platform performance on iOS, Android, and Web.*

## 🎮 How It Works

### **Step 1: Swipe to Like or Skip**
Browse curated property listings with simple swipe gestures. Right for like, left to pass.

### **Step 2: Tap Card for Full Details**
Get complete property information including photos, price, location, amenities, and landlord profiles.

### **Step 3: Chat or Contact Landlord**
When you match, start conversations instantly through our built-in messaging system.

## 🛠️ Technical Stack

- **Framework**: React Native with Expo SDK 53
- **Navigation**: Expo Router with file-based routing
- **State Management**: Custom context hooks + React Query
- **Animations**: React Native Animated API for smooth interactions
- **Storage**: AsyncStorage for offline preferences
- **Icons**: Lucide React Native for consistent iconography
- **Styling**: StyleSheet with theme system
